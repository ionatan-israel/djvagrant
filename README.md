## djvagrant

### Uso:

```shell
django-admin.py startproject --template=https://github.com/jrperdomoz/djvagrant/archive/master.zip --extension=py,conf,rst,bat,yml,rb project_name
```

### Renombrar Vagrantfile
```shell
mv Vagrantfile.rb Vagrantfile
```

### Arrancar maquina
```shell
vagrant up
```
